# Stonks Watchlist

Decision-first equity watchlist — 22 tickers, 6-factor scoring, trade plan, risk sizer.

Live: https://21e8-miner.github.io/stonks/

Single self-contained HTML file (`index.html`). Edits made in the browser persist locally via localStorage; use Export/Import JSON or the Copy Share Link button to move state between people.

Personal research notes, not investment advice.
